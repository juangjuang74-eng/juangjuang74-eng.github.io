# Quantum-Inspired ARC Solver

A novel AI system built for the **ARC Prize 2025** competition on Kaggle.

Instead of committing to a single answer, this solver maintains a **superposition
of candidate solutions** — each assigned a complex amplitude inspired by quantum
mechanics. Candidates evolve through phase rotation, and the best solution
emerges via a hybrid quantum–domain verification collapse.

## Highlights
- 🌌 Quantum hypothesis engine with up to 25 simultaneous candidates
- 🧬 Pattern learner that extracts color & spatial rules from training pairs
- ⚡ Amplitude-based evolution over 15 steps per task
- 🎯 Hybrid scoring: 40% quantum probability + 60% domain verifier
