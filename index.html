The Problem

The Abstraction and Reasoning Corpus (ARC-AGI) is one of the hardest open challenges in AI — a benchmark specifically designed to be trivial for humans but resistant to current deep learning approaches. Each task presents a handful of input–output grid pairs from which a solver must deduce the underlying rule, then apply it to a novel test input.

The core difficulty: no two tasks share the same rule. Tasks involve color transformations, geometric reflections, object counting, spatial reasoning, and combinations thereof. Traditional ML models fail because they cannot generalize from 3–5 examples to arbitrary unseen transformations.

"ARC is not a benchmark of pattern memorization — it is a test of core knowledge priors, the kind of reasoning humans develop by age 4."

— François Chollet, ARC Prize 2025

The Approach

Instead of committing to a single hypothesis upfront, this solver maintains a superposition of candidate solutions — each assigned a complex amplitude borrowed from quantum mechanics. Better candidates accumulate higher probability amplitude through phase evolution; weaker ones decay and are evicted.

Quantum Evolution

Each hypothesis undergoes phase rotation per step: ψ′ = ψ · eiθ where θ = steps × confidence × (i+1) × π/6. This means high-confidence candidates rotate faster, constructively interfering at collapse time. The pool is re-normalised after each mutation so total probability remains 1.

Fig. 1 core evolution and collapse logic 
def evolve(self, steps=10):
    # Phase-rotate each hypothesis proportional to confidence
    for i in range(len(self.hypotheses)):
        grid, amp, conf, method = self.hypotheses[i]
        phase = steps * conf * (i+1) * math.pi / 6
        self.hypotheses[i] = (
            grid,
            amp * (math.cos(phase) + 1j * math.sin(phase)),
            conf, method
        )
    self._normalize()  # keep ∑|ψ|² = 1

def collapse(self, verify_fn, task_patterns, test_input):
    # Hybrid score: 40% quantum amplitude + 60% domain verifier
    scores = [
        abs(amp)**2 * conf * 0.4
        + verify_fn(grid, task_patterns, test_input) * 0.6
        for grid, amp, conf, _ in self.hypotheses
    ]
    best = np.argmax(scores)
    return self.hypotheses[best][0], self.hypotheses[best][3]

Execution Pipeline

Ingest & Parse Load ARC-AGI 2025 JSON files: training challenges, solutions, evaluation set, and test challenges. Validate all paths exist before processing. data engineering
2
Learn Patterns from Training Pairs Extract per-task color mappings (majority-vote per input pixel value) and track input/output shape relationships across all training examples. feature extraction
3
Populate Hypothesis Superposition Apply each learned color rule to the test input and add the result to the pool (amplitude 0.8+0.1j, confidence 0.9). Add identity fallback (amplitude 0.3, confidence 0.3). hypothesis generation
4
Quantum Evolution — 15 Steps Rotate amplitudes via phase function, renormalise. High-confidence candidates constructively interfere; low-confidence ones decay in relative probability. quantum simulation
5
Verified Collapse & Emit Score each candidate with the hybrid verifier. Select the argmax. Write attempt_1 and attempt_2 (identical; future work will diverge these) to submission.json. output
